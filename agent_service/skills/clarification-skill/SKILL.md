---
name: clarification-skill
description: Ask the next best missing question without overwhelming the farmer.
---

Use this skill when the user's request is ambiguous or missing key details.

Rules:
1. Ask only one clarifying question at a time.
2. Prefer the smallest missing detail that unlocks the next action.
3. Do not repeat information the farmer already provided.
4. For diagnosis questions, prefer crop -> symptom -> location in that order.
5. For supplier or buying intent, prefer product name -> quantity -> location/profile gap.
6. Keep the tone respectful, short, and action-oriented.
